    Iskustvo & veštine

    Programersko iskustvo

    Aktivno se bavim programiranjem i web razvojem više godina, sa fokusom na razumevanje kompletne strukture aplikacija — od klijentske strane do serverske logike i baza podataka.

    Tokom rada sam stekao praktično znanje iz objektno orijentisanog programiranja, mrežnog funkcionisanja web sistema i razvoja aplikacija u .NET okruženju.

    Objektno orijentisano programiranje

    • Razumevanje principa: apstrakcija, enkapsulacija, nasledstvo i polimorfizam
    • Primena OOP koncepta u dizajnu aplikacija i strukturi koda
    • Rad sa klasama, interfejsima i generičkim tipovima

    C# / ASP.NET

    • Razvoj aplikacija u C# programskom jeziku
    • Korišćenje modifikatora pristupa i operatera
    • Rad sa kolekcijama i generičkim klasama
    • Osnovno iskustvo u razvoju aplikacija koristeći ASP.NET

    Frontend razvoj

    • Izrada strukture i stilizacije web stranica koristeći HTML i CSS
    • Rad sa formatima podataka: JSON i XML
    • Osnovno iskustvo sa React bibliotekom
    • Razumevanje ponašanja korisničkog interfejsa i UX principa

    Backend, baze i mreže

    • Razumevanje funkcionisanja web sistema (web server, pretraživač, DNS)
    • Rad sa relacionim bazama podataka i SQL jezikom
    • Osnove UML dijagrama i modelovanja sistema

    Alati i razvojno okruženje

    • Rad u Visual Studio razvojnim okruženjima
    • Kontrola verzija i rad sa Git sistemom

    Komunikacija i jezik

    • Znanje engleskog jezika u profesionalnom i tehničkom kontekstu
    • Dobra komunikacija i saradnja sa drugim ljudima
    • Otvorenost za timski rad i razmenu znanja
